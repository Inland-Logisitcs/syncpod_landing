# Chat Application

## Overview

This is a real-time chat application built with React, Vite, and Tailwind CSS. It allows users to sign up, log in, and chat with each other.

## Features

*   User authentication (login and signup)
*   Real-time messaging
*   A clean and modern user interface

## Design

*   **Framework**: React with Vite
*   **Routing**: React Router
*   **Styling**: Tailwind CSS
*   **Component-Based Architecture**: The application is divided into reusable components for easy maintenance and scalability.

## Implemented Changes (Migration from Next.js to Vite)

*   **Removed Next.js**: All Next.js related dependencies and configurations have been removed.
*   **Integrated Vite**: The project now uses Vite for a faster development experience.
*   **Added React Router**: For client-side routing.
*   **Created Application Pages**: The following pages have been created:
    *   **HomePage**: The landing page of the application.
    *   **LoginPage**: The page where users can log in.
    *   **SignupPage**: The page where users can create a new account.
    *   **ChatPage**: The main chat interface.
